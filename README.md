# C端诊断工具

## 安装运行环境
```
//安装node
brew install node
//检测版本
node -v
//安装vue开发环境： 
npm install --global vue-cli
//安装cnpm:  
npm install -g cnpm --registry=https://registry.npm.taobao.org
```
## 下载代码
```angular2
//获取项目
git clone git@github.com:erha367/ntcs_tools.git
//进入目录
cd ntcs_tools
//安装依赖
cnpm install
```
## 运行代码
```angular2
//运行代码
npm run serve
```
